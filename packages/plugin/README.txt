== Installation ==

1. Upload the plugin files to `/wp-content/plugins/cloudstudio-elementor-widgets`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Make sure Elementor is installed and activated
4. Start using CloudStudio widgets in Elementor!

== Frequently Asked Questions ==

= Does this plugin require Elementor Pro? =

No, the plugin works with the free version of Elementor. However, it's optimized for Elementor Pro.

= Is this compatible with my theme? =

Yes! The plugin works with any WordPress theme that supports Elementor.
